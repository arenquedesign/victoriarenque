Portfolio — 2024

An editorial system built on restraint and precision. The portfolio operates through extreme typographic contrast — Oswald's geometric austerity set against Roboto's quiet legibility — held together by a single warm cream field that breathes between every element. Motion is choreographic: every transition is timed not to impress, but to guide. The cursor itself becomes an active design element, inverting the world it touches through mix-blend-mode: difference. Smooth scroll carries a deliberate inertia, slowing the eye before content arrives. Interactions are cinematographic in their easing — expo.out for entrances, expo.in for exits — creating the sensation of weight and momentum. Nothing moves without reason; everything that moves does so with intention.

05

Components

Eight discrete components, each with a single defined responsibility. No component duplicates the behavior of another. States are explicit and exhaustive.

Component 01
Cursor Dot

A 20px white circle that follows the mouse with a deliberate 0.15s lag. Uses mix-blend-mode: difference to invert all colors it touches — functioning as both cursor and active visual element. On hover of interactive elements, expands to 48px with a smooth ease.

Default — 20px Hovering — 48px mix-blend-mode: difference
Size default: 20px × 20px · border-radius: 50%
Size hover: 48px × 48px
Color: white (inverted via blend mode)
Position: fixed, z-index: highest layer
Transition: GSAP quickTo, duration 0.15s, power2
Component 02
Header

Fixed to the top of the viewport across all pages. Contains three elements: avatar (clickable, opens modal) left-aligned; "AVAILABLE FOR WORK" label centered, which hides on scroll down and letter-reveals on scroll up; navigation menu toggle right-aligned.

Default Scrolled down — text hidden Scrolled up — text visible
Position: fixed top: 0, width: 100%
Center text: Roboto 400 · uppercase · letter-spacing 0.1em
Scroll threshold: 50px before text hides
Animation: per-letter fade + slide
Component 03
Navigation Menu

Full-screen overlay triggered by the header toggle. Three layered panels slide in from the right with staggered expo.out timing. Navigation links appear with a cascading fade-up stagger. Dismisses on ESC key or navigation link click.

Closed Opening — panels stagger Open — links visible Closing — reverse
Panels: 3 · slide from right · expo.out 0.8s · stagger 0.15s
Links: fade + slide up · power3.out 0.6s · stagger 0.08s
Dismiss: ESC key or link click
Close animation: timeline.reverse()
Component 04
Project List Item

A horizontal row in a 4-column grid: year / title / company / skills. On container hover, all sibling rows dim to opacity 0.3 while the hovered row stays at 1. Simultaneously, the page background transitions to the project's theme color. Dividers dim to translucent.

Default — all visible Spotlight active — siblings 0.3 This item — opacity 1.0
Grid: 4 columns — year / title / company / skills
Spotlight transition: opacity 0.3s ease-in-out
Background shift: body.backgroundColor · 0.6s ease
Dividers: navy → translucent on spotlight
Row gap: 30px

Avatar Modal

A centered lightbox that appears on avatar click. Contains a circular portrait photo. The backdrop uses blur. Entry and exit use a scale + opacity GSAP animation. Closes on ESC key or overlay click.

A pill-shaped toggle with two options: "Personal" and "Real cases". The active state fills the pill background with navy. Currently not visible in the interface (display: none) — reserved for a future filter feature.

06

Micro­interactions

Thirteen distinct interaction moments. Each one is a contract between the interface and the user — an answer to an action, with precise timing and intent. None of them are decorative; all of them are functional.

01 Cursor Dot Tracking
Trigger mousemove anywhere on the page
Values GSAP quickTo · 0.15s · power2
Smooth positional lag — deliberate, not incidental
02 Cursor Expand on Hover
Trigger mouseenter on any interactive element (links, buttons, project rows)
Values 20px → 48px · 0.3s · power2.out
Reverses on mouseleave
03 mix-blend-mode Inversion
Trigger cursor overlaps any element
Values mix-blend-mode: difference (CSS, always active)
White cursor inverts underlying colors — text, images, backgrounds
04 Project Spotlight
Trigger mouseenter on project list container
Values All rows: opacity → 0.3
Hovered row: opacity → 1.0
Transition: 0.3s ease-in-out
05 Background Color Shift
Trigger mouseenter on specific project row
Values body.backgroundColor → project theme color
transition: 0.6s ease
Resets to #FFFEF8 on mouseleave
06 Lenis Smooth Scroll
Trigger any scroll event (wheel, touch, keyboard)
Values lerp: 0.1 · wheelMultiplier: 0.7
Page momentum lingers — sensation of mass in motion
07 Menu Open
Trigger click on header toggle button
Values 3 panels: slide right → in · expo.out · 0.8s · stagger 0.15s
Links: fade + slide up · power3.out · 0.6s · stagger 0.08s
08 Menu Close
Trigger click outside, ESC key, or navigation link click
Values GSAP timeline.reverse()
Panels exit in reverse stagger order
09 Header Text Reveal / Hide
Trigger scroll down > 50px → hide
scroll up → reveal
Values Per-letter animation — hide: slide up + fade
Reveal: staggered slide down + fade in
Creates textual presence that responds to reading intent
10 Case Sheet Open
Trigger click on CRM project row
Values translateY(100%) → 0 · expo.out · 0.8s
Secondary Lenis instance initializes on complete
11 Case Sheet Close
Trigger click X button, ESC key
Values 0 → translateY(100%) · expo.in · 0.65s
Scroll resets to 0 · Secondary Lenis destroyed
12 Close Button Inversion
Trigger sheet content scrolls past header zone
Values Navy background → cream background
Cream text → navy text
Maintains legibility across both dark and light zones
13 Video Autoplay Loop
Trigger page load / sheet open
Values autoplay · loop · muted · playsinline
.mov format · no controls · functions as animated gif
Seamless, unattended demonstration
